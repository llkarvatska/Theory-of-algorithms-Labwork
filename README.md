# Лабораторна робота №6
**Дисципліна:** Теорія алгоритмів  
**Тема:** Алгоритм Дейкстри  
**Виконала:** Карвацька Єлизавета, АІ-241

## Завдання
Знайти найкоротші шляхи в заданому зваженому графі від вершини **1** за допомогою алгоритму Дейкстри.

## Реалізація
Програма реалізована мовою **Python** з використанням:
- пріоритетної черги `heapq`,
- словників `dist` (мінімальні відстані) та `pred` (попередники).

Алгоритм поетапно оновлює мінімальні відстані для всіх вершин графа.
