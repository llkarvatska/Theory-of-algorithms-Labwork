##  Завдання

##  Реалізація
У програмі реалізовано алгоритм **HeapSort** мовою **Python**.  
Використано дві основні функції:
- `sink()` — занурення елемента у купу;
- `heapsort()` — побудова максимальної купи та послідовне вилучення найбільших елементів.
